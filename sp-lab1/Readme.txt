For sp-lab1.c, it has five options for user to select. 1. Print the Database. 2. Lookup employee by ID. 3. Lookup employee by last name. 4. Add an Employee. 5. Quit.

You can select what fuction you want to do.

At first, the sp-lab1.c will read the file using readfile.c and readfile.h, and store the data in the array, then do the fuction that user want to do until enter 5 to quit.

To exeucte the sp-lab1.c, needs to use the commend /sp-lab1 inputfile.txt to execute the program, so that the sp-lab1.c can read the file and execute successfully.
